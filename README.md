## Project
A little Opengl Voxel World engine

We used `glfw3` API for creating windows and manage input and events,
and `glm` to manage matrix/vector/quaternion.

## Installation

- Install `glfw` with brew

	```brew install glfw3```

- Install `glm` with brew

	```brew install glm```

- Add precommit rules

	```make init```

- Compile the project

	```make -j8```
- Run the project

	```./ft_vox```
