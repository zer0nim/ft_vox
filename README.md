## Project
A little Opengl Voxel World engine

We used `glfw3` API for creating windows and manage input and events,
and `glm` to manage matrix/vector/quaternion.

## Installation

- Install depencies (osx)

	```./configure```

- Add precommit rules

	```make init```

- Compile the project

	```make -j8```
- Run the project

	```./ft_vox --usage```
	```./ft_vox --name <map_name>```
